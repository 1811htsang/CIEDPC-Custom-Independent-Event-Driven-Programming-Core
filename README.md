# CIEDPC-Custom-Independent-Event-Driven-Programming-Core

This repository aims to leverage the core logic of the event-driven programming model and existing AKEDP implementations designed by EPCB-vn to create a new module – CIEDPC – that is customizable and hardware-independent for use on various embedded development kits.

Currently, CIEDPC focuses on optimizing size and hardware independence for unit testing before being used on embedded development kits such as STM32 (STM32F103) and ESP32 (ESP32-WROOM32/ESP32-S3).

Note that CIEDPC is designed with core features essential to the event-driven programming model before planning for functionality or logic expansion.
